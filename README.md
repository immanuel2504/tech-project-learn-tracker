# tech-project-learn-tracker

Personal learning and task tracker for technical projects — built to learn by doing.

## Projects included

- **Food Mfg IoT** — production tracking, machine health, AI prediction, dashboards
- **Crop Disease AI** — image classification, ML API, alerts, agronomist dashboard

## Features

- Task tracking by learning phase
- **Habit Builder** — daily learning habits, streaks, 7-day progress
- Learning notes (saved in browser)
- Architecture reference diagrams
- Glossary per project
- **Advisor Mindset** — Trusted Advisor + Partner Mindset for solution design

## Live app

**https://immanuel2504.github.io/tech-project-learn-tracker/**

(GitHub Pages — may take 1–2 minutes after each push to update.)

## How to run locally

### Option 1 — No server (easiest)

Double-click `OPEN.bat` or open `index.html` in your browser.

### Option 2 — Local server

Double-click `start.bat`, then open:

```
http://localhost:8888
```

## Tech stack

- HTML / CSS / JavaScript
- localStorage for saving progress (no backend required)
